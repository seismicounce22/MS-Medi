Pharmacy Inventory Management System (C++ | DSA Project)
 Overview

The Pharmacy Inventory Management System is a console-based application developed in C++ that simulates real-world medical store operations.

This project was built to apply core Data Structures & Algorithms concepts in a practical system rather than just solving isolated problems.

It allows efficient management of medicines, sales tracking, and issue handling.

Features

Add new medicines to inventory

Search medicines by name

Display complete inventory

Sell medicines and update stock

Update medicine details

Delete medicines

View recent sales records

Manage issue queue
